# 무명: 이름 없는 병사 (Nameless: The Unnamed Soldier)

천계와 심연의 대립, 그리고 그 이면의 진실을 탐구하는 인터랙티브 텍스트 RPG 웹 애플리케이션입니다.

---

## 🎮 게임 특징
- **선택지와 멀티 엔딩**: 플레이어의 신앙(Faith), 의심(Doubt), 상처(Wound), 기억(Memory) 스탯에 따라 대화와 사건이 분기되며 4가지 최종 엔딩과 여러 조기 종료로 연결됩니다.
- **실시간 스탯 & 단서 시스템**: 플레이어의 선택 결과와 수집한 세계관의 단서가 화면 상단에 실시간으로 시각화되어 몰입감을 높입니다.
- **체크포인트 불러오기**: 매 갈림길 선택 시 자동으로 진행 상황이 저장되며, 언제든 최근 선택 시점으로 불러오기(↺)가 가능합니다.
- **미니멀리즘 다크 디자인**: 분위기 있는 radial 그라데이션 백그라운드와 시적인 텍스트가 자연스럽게 흘러가는 폭포식 마이크로 애니메이션을 적용했습니다.

## 📖 스토리 및 시스템 업데이트 내역 (v1.1)
최근 업데이트를 통해 각 막의 서사 개연성과 시스템적 깊이가 한층 보강되었습니다.
- **1막 (천계)**: 벽화의 조사 여부 및 레오와의 공유 여부에 따른 성역 붕괴 직전의 디테일한 묘사(`touched_mural`, `shared_mural_with_leo` 플래그)와 보관소 기록 획득 분기가 다듬어졌습니다.
- **2막 (심연)**: 안내자를 거부하고 혼자 길을 헤매다 탈진하여 결국 합류하게 되는 서사적 개연성 노드(`act2_008_refused`)가 신설되었습니다.
- **3막 (조작)**: 최초의 성채 기록실에 진입할 때의 선택지(열쇠 강탈, 사제 쪽지 제시, 쌍둥이 봉인 간파)에 따른 전용 묘사 노드들이 추가되었습니다.
- **4막 (정산)**: 오르덴의 고백 후 바르카스와의 대면이 플레이어의 성향(의심/신앙 스탯)에 따라 서로 다른 대치 상황(`왕좌 문턱`)으로 전개되며, 그의 최후와 유언을 전해 듣고 기록하는 단계가 유기적으로 연결되었습니다.
- **최종막 (원초의 정원)**: "모든 사실 폭로" 및 "왕좌 거부(진 엔딩)" 선택 시, 충분한 증거(단서)나 스탯(기억, 의심)이 부족할 경우 개연성 있게 실패하여 차선책으로 이어지는 완충 연출 노드(`final_truth_unravel`, `final_true_unready`)가 보강되었습니다.

## 🛠️ 기술 스택
- **프론트엔드**: Vanilla HTML5, Vanilla CSS3, Vanilla JavaScript (ES Module이 아닌 file:// 로컬 실행 최적화 구조)
- **배포**: GitHub Pages (자동 호스팅)

## 🚀 실행 방법
이 저장소는 GitHub Pages로 설정되어 있어 [https://gapfixnid.github.io](https://gapfixnid.github.io) 주소에서 웹 브라우저로 접속해 즉시 플레이하실 수 있습니다. 로컬에서 실행 시 `index.html` 파일을 더블클릭하면 즉시 구동됩니다.
